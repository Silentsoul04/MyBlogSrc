jekyll mac 安装
Aug 5, 2017

升级mac自带的rubby

$ curl -L get.rvm.io bash -s stable

$ rvm -v

$ rvm list known

$ rvm install 2.4.0

安装jekyll

$ git clone git://github.com/jekyll/jekyll.git

$ cd jekyll

$ script/bootstrap

$ bundle exec rake build

$ ls pkg/*.gem head -n 1 xargs gem install -l

参考：https://jekyllrb.com/docs/installation/
